# 모바일 QR 코드 접속

## 작동 원리

Glasshouse는 로컬에서 HTTP 서버를 시작하고 **LAN 주소**(예: `http://192.168.1.100:7008`)를 생성합니다. QR 코드를 스캔하면 같은 WiFi 네트워크를 통해 휴대폰에서 Claude Code를 사용할 수 있습니다.

## 연결할 수 없는 경우

1. **같은 네트워크가 아님** — 휴대폰과 컴퓨터가 같은 WiFi에 연결되어야 합니다 (같은 라우터/같은 네트워크 이름)
2. **방화벽 차단** — OS 방화벽이 수신 연결을 차단할 수 있습니다
3. **기업 네트워크 격리** — AP 격리로 장치 간 통신이 차단될 수 있습니다
4. **VPN 간섭** — VPN이 네트워크 경로를 방해할 수 있습니다

## 보안 참고사항

> ⚠️ Glasshouse의 LAN 서비스는 같은 네트워크의 모든 장치에서 접근 가능합니다.

- **공용 WiFi**에서 사용 시 주의하세요
- Glasshouse는 **토큰 인증**으로 LAN 접근을 보호합니다
- 신뢰할 수 있는 네트워크에서 사용을 권장합니다

## LAN 외부 접속

- **터널링 도구** — frp, ngrok, Tailscale 등
- **Glasshouse 플러그인** — 플러그인으로 프록시를 구성하여 네트워크 간 접근 가능
