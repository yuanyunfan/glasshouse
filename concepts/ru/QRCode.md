# Мобильный доступ по QR-коду

## Как это работает

Glasshouse запускает HTTP-сервер и генерирует **адрес в локальной сети** (например, `http://192.168.1.100:7008`). Отсканируйте QR-код, чтобы получить доступ к Claude Code с телефона через ту же сеть WiFi.

## Почему не удаётся подключиться?

1. **Не в одной сети** — Телефон и компьютер должны быть подключены к одному WiFi (один роутер/одно имя сети)
2. **Блокировка файрвола** — Файрвол системы может блокировать входящие соединения
3. **Изоляция корпоративной сети** — Изоляция AP может препятствовать связи между устройствами
4. **Помехи VPN** — VPN может нарушить сетевой маршрут

## Уведомление о безопасности

> ⚠️ LAN-сервис Glasshouse доступен всем устройствам в одной сети.

- Будьте осторожны в **публичных WiFi**
- Glasshouse использует **аутентификацию по токену** для защиты LAN-доступа
- Рекомендуется использовать в доверенных сетях

## За пределами LAN

- **Инструменты туннелирования** — frp, ngrok, Tailscale и др.
- **Плагины Glasshouse** — Настройте прокси-сервер через систему плагинов
