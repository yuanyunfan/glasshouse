# 手機掃碼程式設計

## 工作原理

Glasshouse 在本機啟動 HTTP 服務後，會生成一個**區域網路位址**（如 `http://192.168.1.100:7008`）。手機掃描 QR Code 後，透過同一 WiFi 網路存取該位址，即可在手機上操作 Claude Code。

## 為什麼掃碼後無法連線？

1. **不在同一區域網路** — 手機和電腦必須連接到同一個 WiFi（同路由器/同網路名稱）
2. **防火牆攔截** — macOS/Windows 防火牆可能阻擋了入站連線
3. **企業/學校網路隔離** — 部分企業網路會隔離裝置間的直接通訊（AP 隔離）
4. **VPN 干擾** — 電腦或手機開啟 VPN 可能導致網路不通

## 網路安全提示

> ⚠️ Glasshouse 的區域網路服務預設對同一網路內的所有裝置開放。

- 在**公共 WiFi** 下使用時請注意安全
- Glasshouse 使用 **Token 驗證**保護存取（URL 自動附帶 token）
- 建議在可信的家庭或辦公網路中使用

## 突破區域網路限制

- **內網穿透工具** — 如 frp、ngrok、Tailscale 等
- **Glasshouse 外掛** — 透過外掛機制配置中間代理，實現跨網路存取
