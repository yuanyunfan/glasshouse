# モバイルQRコードアクセス

## 仕組み

Glasshouse はローカルで HTTP サーバーを起動し、**LAN アドレス**（例：`http://192.168.1.100:7008`）を生成します。QR コードをスキャンすると、同じ WiFi ネットワーク経由でスマートフォンから Claude Code を操作できます。

## 接続できない場合

1. **同じネットワークにいない** — スマートフォンと PC が同じ WiFi に接続されている必要があります（同じルーター/同じネットワーク名）
2. **ファイアウォール** — OS のファイアウォールが着信接続をブロックしている可能性があります
3. **企業ネットワーク隔離** — AP 隔離により端末間通信が遮断されている場合があります
4. **VPN 干渉** — VPN がネットワーク経路を妨げている可能性があります

## セキュリティに関する注意

> ⚠️ Glasshouse の LAN サービスは、同じネットワーク上のすべてのデバイスからアクセス可能です。

- **公共 WiFi** では注意してください
- Glasshouse は **トークン認証**で LAN アクセスを保護します
- 信頼できるネットワークでの使用を推奨します

## LAN を超えたアクセス

- **トンネリングツール** — frp、ngrok、Tailscale 等
- **Glasshouse プラグイン** — プラグインでプロキシを設定し、ネットワークを越えてアクセス可能
