# เข้าถึงผ่าน QR Code บนมือถือ

## วิธีการทำงาน

Glasshouse เริ่มเซิร์ฟเวอร์ HTTP และสร้าง**ที่อยู่ LAN** (เช่น `http://192.168.1.100:7008`) สแกน QR Code เพื่อเข้าถึง Claude Code จากมือถือผ่านเครือข่าย WiFi เดียวกัน

## ทำไมเชื่อมต่อไม่ได้?

1. **ไม่ได้อยู่ในเครือข่ายเดียวกัน** — มือถือและคอมพิวเตอร์ต้องเชื่อมต่อ WiFi เดียวกัน (เราเตอร์เดียวกัน/ชื่อเครือข่ายเดียวกัน)
2. **ไฟร์วอลล์บล็อก** — ไฟร์วอลล์ของระบบอาจบล็อกการเชื่อมต่อขาเข้า
3. **การแยกเครือข่ายองค์กร** — AP Isolation อาจป้องกันการสื่อสารระหว่างอุปกรณ์
4. **VPN รบกวน** — VPN อาจรบกวนเส้นทางเครือข่าย

## คำเตือนด้านความปลอดภัย

> ⚠️ บริการ LAN ของ Glasshouse สามารถเข้าถึงได้จากทุกอุปกรณ์ในเครือข่ายเดียวกัน

- ระวังเมื่อใช้ **WiFi สาธารณะ**
- Glasshouse ใช้ **การยืนยันตัวตนด้วยโทเค็น** เพื่อปกป้องการเข้าถึง LAN
- แนะนำให้ใช้ในเครือข่ายที่เชื่อถือได้

## นอกเหนือจาก LAN

- **เครื่องมือ Tunneling** — frp, ngrok, Tailscale เป็นต้น
- **ปลั๊กอิน Glasshouse** — กำหนดค่า proxy middleware ผ่านระบบปลั๊กอิน
